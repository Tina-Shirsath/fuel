## 🚗 1. Fuel Economy Analysis & Prediction

### Objective

Analyze vehicle data (2008–2018) and build predictive models.

### Tools

Python, Pandas, Matplotlib, Scikit-learn

### Key Work

* Data cleaning and merging
* Visualization of fuel efficiency trends
* Clustering using K-Means
* Classification (Logistic Regression, Decision Tree)
* Regression to predict City MPG

### Conclusion

Fuel efficiency improved over time and machine learning models successfully predicted vehicle performance.
